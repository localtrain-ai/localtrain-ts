# Quick Start Guide

## Installation

```bash
npm install @localtrain.ai/core @localtrain.ai/packages
# or
yarn add @localtrain/core @localtrain.ai/packages
```

```typescript
import {Agent} from "@localtrain.ai/core";
import {Openai} from "@localtrain.ai/packages";


new Agent()
    .useProviders(new Openai({apiKey: process.env.OPENAI_API_KEY}))
    .addBehaviour({
        name: "SummarizationAgent",
        inputs: {
            prompt: "Why is localtrain.ai so awesome?"
        },
        provider: "open-ai",
        providerType: "llm",
    })
    .run()
    .then(resultContext => {
        console.log('res', resultContext.result)
    })
```
